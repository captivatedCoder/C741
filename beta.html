Beta Distribution
Models: Probability for proportions; flexible model for situations where possible values are in a a range (generally 0 to 1).
Example: The percentage of the day that will be sunny.
Notation: X ~ Beta\((\alpha, \beta) \)
Parameters: \(\alpha, \beta\) are shape parameters, both must be greater than 0. Sometimes additional parameters a and b are included if the range of values modeled is from a to be rather than 0 to 1.
Possible values: All values between 0 and 1 (or a and b if additional parameters are added)
Mean and variance: E(X) = \(\alpha \over \alpha+\beta\), Var(X) = \(\alpha * \beta \over (\alpha+\beta)^2*(\alpha+\beta+1)\)


Beta distribution is the representation of all the possible values of a probability when we don't know what that probability is.
