Chapter 2

Explain and identify random variables, discrete variables, continuous variables, distributions, confidence intervals, and hypothesis testing

Term Definition Example
Random Variables
Random Variable A rule that assigns a number to every outcome in the sample space of an experiment. In a coin toss experiment, a random variable may assign 1 to heads and 0 to tails.
Discrete Random Variable A random variable with a countable number or distinct values. All integers between 0 and 100.
Continuous Random Variable A random variable representing any value with a range of values. All real numbers between 0 and 100.
Distributions
Probability Mass Function (pmf) Assigns the probability that a discrete random variable is exactly equal to some value. In recent years, 2% of the adult population of the U.S. is classified as underweight, 27% as normal weight, 33% as overweight, and 38% as obese. The pmf of W is then p(W=0) = 0.02, p(W=1) = 0.27, p(W=2) = 0.33 and p(W=3) = 0.38. The probabilities assigned in a pmf are between 0 and 1, and the total probability must sum to 1.
Cumulative Distribution Function (cdf) The probability that for any number x, the observed value of the random variable will be at most x or p(X≤x) When a fair die is rolled and the value facing up recorded, the cdf describes the probability of getting less than or equal to any value x, such as the probability X is less than or equal to 3; p(X â‰¤ 3) = p(1) + p(2) + p(3).
Probability Density Function (pdf) Describes the likelihood of all values for a continuous random variable The amount of time for Casey to do his chores is a random variable, X, where all values between 1 hour and 2 hours are equally likely.
Binomial Distribution Models how many times an event occurs in a certain number of trials. A basketball player is fouled 3 on a 3 point shot attempt and awarded 3 free throws. This models the number of free throws the player makes.
