C741 STATISTICS FOR DATA ANALYTICS STUDY RESOURCE


I wanted an easier way to view the information from the course so I created this website. The Models page contains not only the information in the book description (Models, Notation, etc.) but also contains important (to me) notes from the book as well as links to external resources obtained from Google searches or course documentation.


If anyone is looking at this and is curious about putting mathematical equations into HTML code, I mostly used MathJax which lets you use TeX, LaTEX, and MathML markup.
