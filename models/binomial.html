Name Binomial Distribution
Definition How many times an event occurs in a certain number of trials.
Formula \(n \choose k \) \(p^k(1-p)^{n-k}\)
Input Values n: Number of trials, p: Probability of the event occuring on each trial, k: The number of successes
Output values Probability of receiving a specific outcome expressed as a fraction or percentage.
Interpretation Written P(x) which means "the probability of x".
Mean and variance: \(\mu\) = E(X) = n * p, \(\sigma^2\) = Var(X) = n * p * (1 - p)
