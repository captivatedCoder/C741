Name Wilcoxon Rank Sum Test / Mann-Whitney Test
Definition The Wilcoxon rank sum test is a nonparametric approach to establishing significant difference between two sample groups using magnitude-based ranks.
Hypotheses \( H_0 : \mu = \mu_0 \)
\( H_a : \mu \ne \mu_0 \)
Formulas Mean \( \mu_w \) = \( n_1(n_1+n_2+1) \over 2 \)

Std Dev σ w = n1 n2 ( n1 + n2 + 1 ) 12
Input Values Categorical (can be quantitative data or ordinal data as it will be converted to categorical)
\( \mu_w \) \( \sigma_w \)
Output Values z value - The difference between the groups divided by the standard deviation. A negative z-score would indicate a statistically significant difference between the group means.
p-value - NORM.S.DIST function in Excel. Value from the normal distribution table, cumulative.
Interpretation of Output If p-value < α reject \( H_0 \)
If one median value is smaller or larger reject the null hypothesis.
Value of the Method Allows for comparison of two populations with distributions that are not normal but have a similar shape. Converts the data into ranks so it can use quantitative or ordinal data. Compares the median of the populations rather than the mean.
Example 10 people were asked to review two stores (5 at each store). There were 5 criteria (Cleanliness, Service, Facilities, Drink Quality, and Food Quality) and a scale of 1-5 (lowest possible score 5, highest possible score 25).
Place samples in two columns. Stack both columns in third, rank this column in the fourth column. Sum the ranks. Use sampling dist and test statistic to find p, test p to alpha level
