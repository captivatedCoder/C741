Name Kruskal Wallis
Definition A nonparametric test that compares more than two non-normally distributed populations using rank sum. Alternate to the ANOVA
Hypothese \( H_0 \): All populations are the same
\( H_a \): All populations are not the same
Formula H= 12 N(N+1) Σ Ri2 ni - 3(N+1)
Input Values N - total observations in all samples
\( R_i \) - sum of all ranks for each sample (i)
\( n_i \) - number of observations in each sample (i)
Output Values Test-Statistic - \( H > \chi^2_a, k-1 \)
P-Value - \( P(\chi^2 > H)\)
Interpretation of Output If the Chi-squared value is less than the H value then reject the null hypothesis
Value of the Method Can be used in place of ANOVA when the data does not fit a normal distribution. Accepts quantitative or qualitative data since all data is converted to ranks. While data does not need to be normally distributed, it does need to be the same shape and have equivalent variance.
Example 45 people are surveyed about 3 stores (15 per store). Asked to rate 5 categories on a scale of 1-5. Steps are similar to Wilcoxon until finding the chi-square value.
